# 📱 AI Lecturer – Flutter Mobile Application

A Flutter-based mobile application developed as part of my Mobile Application Development project. This app demonstrates modern UI design, structured architecture, and AI-based learning features such as slide generation and interactive Q&A.

---

## 🚀 Features

- 📱 Clean and user-friendly interface  
- ⚡ Fast and responsive performance  
- 🤖 AI-based slide generation  
- 🎤 AI voice lecture presentation  
- 💬 Interactive Q&A system  
- 🧩 Modular and maintainable code structure  

---

## 🛠️ Technologies Used

- Flutter (UI Framework)  
- Dart (Programming Language)  
- Android Studio / VS Code
