# 📱 AI Lecturer – Flutter Mobile Application

A Flutter-based mobile application developed as part of my Mobile Application Development project. This app demonstrates modern UI design, structured architecture, and AI-based learning features such as slide generation and interactive Q&A.

---

## 🚀 Features

- 📱 Clean and user-friendly interface  
- ⚡ Fast and responsive performance  
- 🤖 AI-based slide generation  
- 🎤 AI voice lecture presentation  
- 💬 Interactive Q&A system  
- 🧩 Modular and maintainable code structure  

---

## 🛠️ Technologies Used

- Flutter (UI Framework)  
- Dart (Programming Language)  
- Android Studio / VS Code  

---

## 📂 Project Structure
```
my_project/
│
├── android/          # Android native files  
├── ios/              # iOS native files  
├── lib/              # Main application code  
│   ├── screens/      # All app screens  
│   ├── widgets/      # Reusable custom widgets  
│   └── main.dart     # App entry point  
├── test/             # Testing files  
├── build/            # Build output  
├── pubspec.yaml      # Dependencies  
└── README.md         # Documentation  
```

## ▶️ Getting Started

1. Clone the repository  
[git clone https://github.com/your-username/my_project.git ](https://github.com/muzalfa786786786-cmyk/TIT601540-AI-lecture-project.git) 

2. Navigate to project folder  
cd my_project  

3. Install dependencies  
flutter pub get  

4. Run the app  
flutter run
